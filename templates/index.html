SecureAI: Cybersecurity RAG Assistant

An AI-powered web application that answers cybersecurity questions using Retrieval-Augmented Generation, vector search, and a large language model.

Powered by RAG

Project Overview

This assistant retrieves relevant cybersecurity knowledge first, then uses an LLM to generate a grounded answer.

  • Flask backend
  • HTML, CSS, JavaScript frontend
  • ChromaDB vector database
  • HuggingFace embeddings
  • Groq LLM

Ask the Assistant

Answers are generated using retrieved cybersecurity knowledge from the app’s knowledge base.

Future ML Module

Next upgrade: train an intrusion detection model using cybersecurity traffic data, save it with Joblib, and connect it to this web app.
